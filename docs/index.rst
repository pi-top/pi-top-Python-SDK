pi-top Python SDK Reference
=====================================================

.. image:: _static/foundation_kit.png
    :target: https://knowledgebase.pi-top.com/knowledge/foundation-kit
    :alt: Foundation Kit

A simple, modular interface for interacting with a pi-top and its related accessories and components.

About
=====

This SDK is allows the user to easily interact and connect different components of a pi-top. It provides
a Python 3 module called `pitop`, with several submodules that communicate with a particular subsystem inside a pi-top.

This SDK also contains CLI utilities, to interact with your pi-top using the terminal.

This library is included out-of-the-box with pi-topOS.

Ensure that you keep your system up-to-date to enjoy the latest features and bug fixes.
