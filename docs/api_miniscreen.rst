======================================
API - pi-top [4] Miniscreen
======================================

pi-top [4]'s case includes a miniscreen module, composed by a OLED screen and 4 buttons.
On pi-topOS, the OLED and buttons display an interactive menu, with useful information about
the system and networking, but they can also be used for projects.
